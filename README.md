# Задание

1. Написать программу, которая будет генерировать файл вида<br/>
	Число. Строка<br/>
	Пример: <br/>
	2423. GFjdhdfjkKDJfjhDFjkOqweqweDSDrwersdf

	Число в диапазоне [0,10000]<br/>
	Длинна строки в диапазоне [20,100]<br/>
	Файл может быть размером до 200ГБ.<br/>
	На вход программа принимает количестов строк в файле.

2. Написать программу, что способна отсортировать файл полученной программой 1.<br/>
Производительность и потребление памяти для программы критичны.
